# DeepSeek Chat Clone

一个基于Vue.js的现代化聊天应用界面，模拟DeepSeek AI助手的用户体验。

## 项目简介

这是一个单页应用(SPA)，提供了类似DeepSeek AI助手的聊天界面，具有以下特性：

- 🎨 现代化的UI设计
- 🌙 深色/浅色主题切换
- 💬 实时聊天功能
- 📱 响应式布局设计
- 🔄 对话历史管理

## 技术栈

- **前端框架**: Vue 3
- **构建工具**: Vite
- **样式**: CSS3 + Flexbox布局
- **语言**: JavaScript (ES6+)

## 项目结构

```
front/
├── src/
│   ├── components/          # Vue组件
│   │   ├── Sidebar.vue      # 侧边栏（对话列表）
│   │   ├── ChatHeader.vue   # 聊天头部
│   │   ├── ChatArea.vue     # 聊天消息区域
│   │   └── MessageInput.vue # 消息输入框
│   ├── App.vue              # 根组件
│   ├── main.js              # 应用入口
│   └── style.css            # 全局样式
├── index.html               # HTML模板
├── package.json             # 项目配置
├── vite.config.js           # Vite配置
└── README.md                # 项目说明
```

## 功能特性

### 核心功能
- **对话管理**: 创建、选择、删除对话
- **消息发送**: 实时消息发送和显示
- **AI响应模拟**: 智能回复生成
- **主题切换**: 深色/浅色模式

### 界面特性
- **自适应布局**: 支持不同屏幕尺寸
- **侧边栏折叠**: 可折叠的对话列表
- **消息格式化**: 支持代码块、标题等富文本格式
- **滚动优化**: 平滑的消息滚动体验

## 安装与运行

### 环境要求
- Node.js (版本 14 或更高)
- npm 或 pnpm

### 安装依赖
```bash
npm install
# 或使用 pnpm
pnpm install
```

### 开发模式
```bash
npm run dev
```
应用将在 http://localhost:3000 启动

### 构建生产版本
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

## 组件说明

### App.vue
应用根组件，负责：
- 全局状态管理
- 对话数据管理
- 主题切换逻辑
- 布局容器

### Sidebar.vue
侧边栏组件，包含：
- 对话列表显示
- 新建对话按钮
- 对话删除功能
- 折叠/展开状态

### ChatHeader.vue
聊天头部组件，显示：
- 当前对话标题
- 对话统计信息
- 响应式设计

### ChatArea.vue
聊天区域组件，负责：
- 消息列表渲染
- 消息格式化
- 滚动控制
- 思考状态指示器

### MessageInput.vue
消息输入组件，提供：
- 消息输入框
- 发送按钮
- 输入状态管理

## 样式设计

### 布局系统
- 使用Flexbox实现响应式布局
- 侧边栏固定宽度(260px)，主内容区域自适应
- 移动端友好的响应式设计

### 颜色主题
- **浅色主题**: 白色背景，灰色边框
- **深色主题**: 深灰背景，适配暗色模式
- 平滑的主题切换动画

### 响应式断点
- 桌面端: > 768px
- 移动端: ≤ 768px

## 开发说明

- 响应式设计原则
